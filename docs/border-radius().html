@mixin border-corner-radius()

Миксин border-corner-radius() является лишь оберткой над миксином prefixer(), который генерирует свойство border-<сторона>-radius с префиксом для движка WebKit.

Since 4.0.0:

Parameters

$vr: (string)

вверхняя (top) или нижняя (bottom) сторона

$hr: (string)

левая (left) или правая (right) сторона

$radius: $border-radius (int | string)

радиус скругления
